# 🌊 Rising Waters – Flood Prediction System

## 📌 Project Overview

The Rising Waters – Flood Prediction System is an AI/ML-based web application developed to predict flood risk using historical rainfall and weather-related data. The system helps users identify potential flood-prone situations and supports early decision-making to minimize damage and improve public safety.

---

## 🎯 Objectives

- Predict flood risk using machine learning algorithms.
- Provide early flood warnings based on rainfall data.
- Assist authorities and citizens in disaster preparedness.
- Improve the accuracy of flood prediction through data analysis.

---

## ✨ Features

- User-friendly web interface
- Flood risk prediction
- Machine learning-based analysis
- Historical rainfall data processing
- Fast and accurate predictions
- Responsive Flask web application

---

## 🛠️ Technology Stack

- **Frontend:** HTML, CSS
- **Backend:** Python, Flask
- **Machine Learning:** Scikit-learn
- **Dataset:** Flood rainfall dataset (CSV)
- **Model:** Pickle (.pkl)
- **IDE:** Visual Studio Code

## 🚀 Installation

1. Clone the repository

```bash
git clone <repository-url>
```

2. Navigate to the project folder

```bash
cd Rising-Waters-Flood-Prediction
```

3. Install dependencies

```bash
pip install -r requirements.txt
```

4. Run the application

```bash
python app.py
```
